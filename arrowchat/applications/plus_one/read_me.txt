If arrowchat is not installed in its default directory you will need to change index.php in plus_one folder.
Default directory example: domain.com/arrowchat/applications/

If arrowchat is installed in a different location open index.php and find this line
"<g:plusone><img id="center" src="/arrowchat/applications/plus_one/images/ajax-loader.gif"></img></g:plusone>"

Change the src attribute to match your arrowchat installation.
